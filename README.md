# AskNLearn

**AskNLearn** este o platforma dezvoltata in cadrul lucrarii de licenta, sub coordonarea Prof. Univ. Asist. Dr. Vilcea Lucian Alexandru, avand ca tema „Platforma colaborativa pentru studiu de grup si comunicare educationala structurata”.  

Scopul platformei este de a oferi un mediu educational sigur, structurat si adaptat nevoilor elevilor si studentilor, sprijinind procesul de invatare si colaborare academica.  

In prezent, majoritatea studentilor folosesc aplicatii precum WhatsApp, Discord sau Reddit pentru a afla si distribui resurse educationale sau pentru a studia impreuna. AskNLearn isi propune sa centralizeze toate aceste functionalitati de socials intr-un mediu dedicat educatiei, combinand siguranta, colaborarea si invatarea activa intr-o experienta completa si organizata.  

## De ce AskNLearn?

AskNLearn aduce impreuna avantajele unei platforme sociale cu beneficiile unui mediu educational special conceput. Platforma permite elevilor si studentilor sa:  

- invete impreuna intr-un spatiu sigur si structurat;  
- acceseze si partajeze resurse educationale de calitate;  
- interactioneze si colaboreze cu alti utilizatori pentru o invatare mai eficienta;
